# CodeJudge SQL Query Implementation

## Overview

This repository contains SQL query implementations and validation notes for the CodeJudge database system.

The project includes:

- basic retrieval queries
- joins
- aggregation queries
- subqueries
- set logic queries
- SQL reasoning explanations
- output validation summaries

---

# Repository Files

| File | Description |
|---|---|
| queries.sql | All SQL queries |
| query_outputs.md | Sample outputs and validation notes |
| sql_reasoning.md | SQL reasoning explanations |

---

# Topics Covered

- filtering
- joins
- aggregation
- HAVING clause
- subqueries
- LEFT JOIN vs INNER JOIN
- validation logic

---

# Database Context

The queries were designed for the normalized CodeJudge relational database schema containing students, courses, submissions, contests, test cases, and related entities.

# Final Note

The queries follow standard SQL practices and are designed for correctness, readability, and logical validation.
